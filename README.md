#  Deloitte Data Analytics & Forensic Technology Case Study

# Overview:

This project replicates a real consulting analytics engagement completed through a Deloitte virtual experience.


# Problem:

• Identify factory with highest machine downtime

• Identify most failing device types

• Investigate gender pay equality patterns


# Approach:

• Data cleaning

• Calculated metrics

• Tableau dashboard

• Excel classification logic


# Key Insights:

• Factory with highest downtime identified

• Specific machine categories responsible for failures

• Multiple roles flagged for pay inequality


# Tools: 

Tableau, Excel
